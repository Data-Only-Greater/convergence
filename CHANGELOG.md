### Other

- Prepare Version 0.2 ([#3](https://github.com/Data-Only-Greater/convergence/pull/3),

- Add Report Attribute Access ([#2](https://github.com/Data-Only-Greater/convergence/pull/2),
  [`9103d9e`](https://github.com/Data-Only-Greater/convergence/commit/9103d9e1478022e121e89a4929479bba5f63fce3))

This commit adds access to the convergence report metrics by adding a namespace containing the
  attributes to the items of a Convergence object.

This commit also adds full coverage to the interface and functions modules (although the scope of
  testing in the functions module could be improved).

Changes:

* Remove broken logging

* Tidy up try catch for Arithmetic errors

* Add a tree of namespace style attributes to allow easier access to the table data

* Slim down tables module for actually used methods

* Add __getitem__ to Convergence to get the namespaces for the grid triplets and test.

* Add attributes for metrics when using an analytical value

* Test analytical metrics and update README

* Reorder Convergence._get_shared

* Test and fix all edge cases. tables is the only module now lacking test coverage.

* Switch build badges to show master branch only.

- Add Scripting ([#1](https://github.com/Data-Only-Greater/convergence/pull/1),
  [`8123c96`](https://github.com/Data-Only-Greater/convergence/commit/8123c96a665534dc4e1c55739f1573f59921ccc9))

This commit adds easy to use scripting where the Convergence class can be used by adding grids to it
  with the new add_grids method. The results are now calculated instantly and can be retrieved by
  printing the object.
